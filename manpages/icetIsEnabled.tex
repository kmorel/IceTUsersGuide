\begin{Name}{3}{icetIsEnabled}{Kenneth Moreland}{\IceT Reference}{\CFunc{icetIsEnabled} -- query enabled status of an \IceT feature.}

  \mansection{Synopsis}

  \textC{\#include <IceT.h>}

  \begin{Table}{4}
    \textC{IceTBoolean }\CFunc{icetIsEnabled}\textC{(}&\textC{IceTEnum}&\CArg{pname}&\textC{);}
  \end{Table}

  
  \mansection{Return Value}

  Returns \CEnum{ICET\_TRUE} if the feature associated with \CArg{pname} is
  enabled, \CEnum{ICET\_FALSE} $(= 0)$ if the feature is disabled.


  \mansection{Errors}

  \begin{Description}[ICET\_INVALID\_OPERATION]
  \item[\CErrorEnum{ICET\_INVALID\_VALUE}] If \CArg{pname} is not a feature
    to be enabled or disabled.
  \end{Description}


  \mansection{Warnings}

  None.


  \mansection{Bugs}

  The check for a valid \CArg{pname} is not thorough, and thus the
  \CErrorEnum{ICET\_INVALID\_VALUE} error may not always be raised.


  \mansection{Notes}

  A list of valid values for \CArg{pname} is given in the documentation for
  \icetEnable.
