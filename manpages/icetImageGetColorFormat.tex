\begin{Name}{3}{icetImageGetColorFormat}{Kenneth Moreland}{\IceT Reference}{\icetImageGetColorFormat, \icetImageGetDepthFormat -- get the format of image buffers}

  \mansection{Synopsis}

  \textC{\#include <IceT.h>}

  \begin{Table}{5}
    \textC{IceTEnum}&\icetImageGetColorFormat\textC{(}&\textC{const }\CType{IceTImage}&\CArg{image}&\textC{);} \\
    \textC{IceTEnum}&\icetImageGetDepthFormat\textC{(}&\textC{const }\CType{IceTImage}&\CArg{image}&\textC{);}
  \end{Table}

  
  \mansection{Description}

  \icetImageGetColorFormat and \icetImageGetDepthFormat return an entry in
  an enumeration that specifies the data format stored in the given
  \CArg{image}.  This format determines which one of the
  \CFunc{icetImageGetColor} or \CFunc{icetImageGetDepth} functions to use
  and the form of the resulting data.


  \mansection{Return Value}

  \icetImageGetColorFormat returns one of the following values with the
  associated meaning for the format of the stored color data.

  @enum_image_color@

  \icetImageGetDepthFormat returns one of the following values with the
  associated meaning for the format of the stored depth data.

  @enum_image_depth@
