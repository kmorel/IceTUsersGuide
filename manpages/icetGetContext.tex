  \mansection{Description}

  The \CFunc{icetGetContext} function retrieves the handle for the current
  context.  This handle may be stored and set for later use with
  \CFunc{icetSetContext} (assuming the context has not been since
  destroyed).


  \mansection{Return Value}

  A handle for the current context.


  \mansection{Errors}

  None.
