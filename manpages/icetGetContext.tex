\begin{Name}{3}{icetGetContext}{Kenneth Moreland}{\IceT Reference}{\CFunc{icetGetContext} -- retrieves the current context}

  \mansection{Synopsis}

  \textC{\#include <GL/ice-t.h>}

  \begin{Table}{3}
    \CType{IceTContext}\textC{ }\CFunc{icetGetContext}\textC{(}&\textC{void}&\textC{);}
  \end{Table}

  
  \mansection{Description}

  The \CFunc{icetGetContext} function retrieves the handle for the current
  context.  This handle may be stored and set for later use with
  \CFunc{icetSetContext} (assuming the context has not been since
  destroyed).


  \mansection{Return Value}

  A handle for the current context.


  \mansection{Errors}

  None.
