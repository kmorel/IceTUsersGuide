  \mansection{Description}

  \icetImageGetWidth, \icetImageGetHeight, and \icetImageGetNumPixels allow
  you to query the size of an \CArg{image} with respect to the number of
  pixels.  These functions define the buffer size returned by
  \CFunc{icetImageGetColor} and \CFunc{icetImageGetDepth}.


  \mansection{Return Value}

  \icetImageGetWidth returns the number of pixels along the horizontal axis
  of the \CArg{image} and \icetImageGetHeight returns the number of pixels
  along the vertical axis of the \CArg{image}.  \icetImageGetNumPixels is a
  convenience function that returns the total number of pixels in
  \CArg{image} (the width times the height).
