  \mansection{Synopsis}

  \textC{\#include <IceT.h>}

  \begin{Table}{3}
    \textC{IceTBoolean}\textC{ }\CFunc{icetImageIsNull}\textC{(}&\CType{IceTImage}&\CArg{image}\quad\textC{);}
  \end{Table}

  
  \mansection{Description}

  Tests whether \CArg{image} is a null image.  A null image is one that has
  no memory allocated to it.  Null images have no pixels and empty pixel
  formats.

  Null images are created with the \CFunc{icetImageNull} function.


  \mansection{Return Value}

  Returns \CEnum{ICET\_TRUE} if \CArg{image} is a null image,
  \CEnum{ICET\_FALSE} otherwise.
