\begin{Name}{3}{icetGLIsInitialized}{Kenneth Moreland}{\IceT Reference}{\CFunc{icetGLIsInitialized} -- determine if the \IceT \OpenGL layer is initialized}

  \mansection{Synopsis}

  \textC{\#include <IceTGL.h>}

  \begin{Table}{3}
    \textC{IceTBoolean }\CFunc{icetGLIsInitialized}\textC{(}&\textC{void}&\textC{);}
  \end{Table}

  
  \mansection{Description}

  Used to determine whether \CFunc{icetGLInitialize} was called for the
  current \IceT context.  If \CFunc{icetGLIsInitialized} returns false,
  then rendering with the \OpenGL layer will not work.


  \mansection{Return Value}

  Returns \CEnum{ICET\_TRUE} if the \OpenGL layer has been initialized for
  the current context, \CEnum{ICET\_FALSE} otherwise.
